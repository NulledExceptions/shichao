Projects
********

* `uhuf <https://github.com/shichao-an/uhuf>`_

    * CLI tool to get files from the connected GitLab repository using API
    * Implemented in Bash with completion support

* `Homura <https://github.com/shichao-an/homura>`_

    * Python downloader with progress

* `Adium Shell <https://github.com/shichao-an/adium-sh>`_

    * Shell utilities and Python wrapper for Adium

* `Twitter Photos <https://github.com/shichao-an/twitter-photos>`_

    * Full-featured command-line tool to get photos from Twitter accounts

* `SoundMeter <https://github.com/shichao-an/soundmeter>`_

    * Command-line tool to obtain sound power in real time for machines that ship with audio input devices

* `RSA <https://github.com/shichao-an/rsa>`_

    * Simple RSA cryptosystem in C as a project for the course *Applied Cryptography*

* `Python and Parallel Programming <https://github.com/shichao-an/PPP>`_ (`report </_static/ppp.pdf>`_)

    * Compared and analyzed parallelism in CPU-bound, I/O-bound and memory-bound applications implemented with *threading*, *multiprocessing* of the standard library, and OpenMP/Cython

* `GeoPort <https://github.com/shichao-an/GeoPort>`_

    * Django-based geo-social platform that facilitates event-based real-time location sharing
    * APIs of Google Maps and Google Geocoding
